# HR-Analytics-Dashboard

An interactive Power BI dashboard built to analyze employee attrition (16.1%), workforce demographics, salary distribution, job roles, and job satisfaction. The dashboard helps HR teams identify trends, uncover insights, and make data-driven decisions to improve employee retention.

## Key Highlights
- 1470 Employees analyzed
- 237 Attrition cases
- 16.1% Attrition rate
- Insights by age, salary slab, education, gender, and job role

## Features
- Interactive visuals and KPI cards
- Attrition analysis by multiple dimensions
- Job satisfaction insights
- Easy filtering and drill-down exploration

## Tech Stack
Power BI | DAX | CSV Dataset | Data Visualization

## Files Included
- HR_Analytics.csv (dataset)
- HR Analytics Dashboard.pdf (dashboard export)
